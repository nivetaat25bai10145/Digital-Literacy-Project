# Digital-Literacy-Project

## Project Overview
This project is based on the role of a Student Digital Ambassador. It focuses on promoting awareness about digital literacy, including online safety, professional digital presence, communication etiquette, and cybercrime awareness.

## Tasks Included

### Task 1: Infographic 
Created a Digital Literacy Awareness infographic using Canva

### Task 2: Digital Portfolio 
Developed profiles on GitHub, LinkedIn, and Stack Overflow to build a professional online presence

### Task 3: Platforms
Explored HackerRank for coding and created a Google Form quiz to collect and analyze responses
Google Form Link: https://docs.google.com/forms/d/e/1FAIpQLSeDWQWLoy5x20oNp4c4ILzhWwaEFLTENeY0gINvjJF0a4atkA/viewform?usp=header

### Task 4: Communication
Learned professional email writing and digital communication etiquette

### Task 5: Cyber Awareness
Studied UPI fraud, created a scenario, and designed a prevention checklist
